Login as root user:
  - In terminal execute:

    apt-get install sudo


  - Open the file /etc/sudoers and read the instructions to modify it!!


  - As of 18/07/2018 the instructions are to modify it using visudo, so open a terminal and execute:

    visudo


  - Add the following to the "# User privilege specification" section in the file, (same as the root setting already found in the section):

    # User privilege specification
      '
      '
    tu_usuario ALL=(ALL) ALL
